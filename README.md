# Simple C++ Game Engine (In Progress)

A modular, text-based game engine built in C++ to demonstrate core object-oriented programming principles.

## 🚀 Current Features
- **Modular Architecture:** Split into clean structure layouts using `.h` headers and `.cpp` implementation files.
- **Data Encapsulation:** Protected vital player attributes (like health and potion pools) using private variables and strict validation gates (getters/setters).
- **Custom Build Automation:** Structured completely using CMake toolchains.
